What is Hamming Code?

Hamming Code is an error-correcting technique invented by Richard Hamming. It adds extra bits called parity bits to a binary message to detect and correct single-bit errors. These parity bits are placed at positions that are powers of two and are calculated based on specific combinations of data bits.

When a message is received, the parity bits are recalculated and compared with the received ones. If there's a mismatch, the system identifies the exact bit that was corrupted using binary indexing. This allows the receiver to correct the error and recover the original message accurately.
